# Features Overview

The Account Planning module provides a complete workflow for managing strategic account plans. This page gives you a high-level overview of all features.

---

## Main Features

### 1. Account Plans List (Manage)

- View all account plans in a table
- Filter by client, status, BCG model, date, overdue tasks, and more
- Save filters for quick access
- Bulk actions: change status, bulk copy, mass delete
- Export to Excel
- Switch to Kanban view

### 2. Create New Account Plan

- Create plans from scratch or from a template
- Enter client, subject, period, vision, mission
- Add financial data and marketing activities in spreadsheet-like tables
- Use custom fields for your business needs

### 3. Plan View (Tabs)

Each plan has several tabs:

| Tab | Purpose |
|-----|---------|
| **Due Diligence** | Financial data, marketing activities, background |
| **Team Information** | Your team and client team members |
| **Service Ability Offering** | Current services and offerings |
| **Planning** | Objectives, tasks, MindMap, revenue, client status, BCG model |
| **Relations** | Linked projects, invoices, estimates, proposals |
| **Meeting Notes** | Call logs and meeting notes |
| **Competitors** | Competitor tracking with threat levels |
| **Activity** | Activity log for the plan |

### 4. Templates

- Create reusable templates
- Start new plans from a template to save time
- Edit and manage templates

### 5. Kanban View

- View plans as cards in columns (Draft, In Progress, Review, Completed, Archived)
- Drag and drop to change status
- Quick overview of plan distribution

### 6. Relations

- Link plans to **Projects**, **Invoices**, **Estimates**, **Proposals**
- Quick Create Project from a plan
- See linked project tasks and milestones
- Bidirectional sync between plan tasks and Perfex tasks

### 7. Export

- **PDF** – Export a plan as a PDF document
- **Excel** – Export the plans table to Excel (.xlsx)
- **MindMap** – Export the MindMap as PNG or SVG

### 8. Dashboard Widget

- Summary of Draft, In Progress, Completed plans
- Overdue tasks count
- Client status (Red, Yellow, Green)
- Revenue vs plan comparison
- At-risk accounts alert

### 9. Client Portal

- Clients can view their account plans (read-only)
- Request updates from staff
- Mobile-friendly layout

### 10. Settings

- Industry options
- Client portal on/off
- Email reminders for tasks
- Recurring plan periods (auto-copy)
- Default status for new plans
- Webhooks for integrations
- Cron job setup

---

## Workflow Example

1. **Create** a new plan for a key client.
2. Fill in **Due Diligence** (financials, marketing).
3. Add **Team Information** and **Service Ability Offering**.
4. Define **Objectives** and **Tasks** in Planning.
5. **Link** related projects and invoices in Relations.
6. Add **Meeting Notes** and **Competitors** as needed.
7. **Export** to PDF for meetings or reporting.
8. Use **Templates** for similar clients next time.
