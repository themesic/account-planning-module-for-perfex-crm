# Templates

Templates let you create reusable structures for account plans. Instead of entering the same fields every time, you start from a template and adjust as needed.

---

## Why Use Templates?

- **Save time** – Pre-fill common fields (vision, mission, objectives, structure)
- **Consistency** – All plans follow the same structure
- **Best practices** – Capture your team's standard approach

---

## Creating a Template

1. Go to **Account Planning** → **Templates**.
2. Click **+ New** (or **Add template**).
3. Fill in:
   - **Name** – Template name (e.g., "Standard Enterprise Plan")
   - **Subject** – Default subject
   - **Vision** – Default vision text
   - **Mission** – Default mission text
   - **Objectives** – Default objectives
   - **Threat** – Default threats section
   - **Opportunity** – Default opportunities
   - **Criteria to success** – Default success criteria
   - **Constraints** – Default constraints
   - **MindMap (data_tree)** – If you use MindMap structure
4. Click **Save** or **Submit**.

---

## Editing a Template

1. Go to **Account Planning** → **Templates**.
2. Find the template you want to edit.
3. Click **Edit**.
4. Make your changes.
5. Save.

---

## Creating a Plan from a Template

1. Go to **Account Planning**.
2. Click **Create from Template** (or use the Templates page).
3. Select the template.
4. Choose the **Client** and **Period** for the new plan.
5. The plan will be created with the template content.
6. Adjust any fields and save.

---

## When to Use Templates

- **Similar clients** – Same industry or size
- **Recurring periods** – Monthly or quarterly plans with the same structure
- **Standard workflows** – Your team's agreed format

---

## Template vs. Copy

| Method | Use When |
|--------|----------|
| **Template** | You want a generic starting point |
| **Copy / Create next period** | You want to copy an existing plan (with its data) to a new period |
