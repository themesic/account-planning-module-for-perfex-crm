# Features Overview

The Account Planning module provides the following features:

## Due Diligence

Conduct and document client due diligence as part of your account planning process.

## Team Information

Manage team details, roles, and assignments for each account planning project.

## Service Ability Offering

Define and track service offerings relevant to each client or account.

## Project Planning

Plan and track project phases, milestones, and deliverables within the account planning context.

## Strategy MindMaps

Create and visualize strategy mind maps to map out relationships, goals, and action plans.

---

These features work together to provide a comprehensive account planning tool that integrates seamlessly with Perfex CRM.
