The main Account Planning screen shows all your account plans in a table. You can filter, sort, save filters, and perform bulk actions.

---

## Opening the Account Plans List

1. Click **Account Planning** in the sidebar menu (address-book icon).
2. You will see the list of all account plans.

---

## Understanding the Table

The table shows:

| Column | Description |
|--------|--------------|
| **Checkbox** | Select plans for bulk actions |
| **ID** | Plan number |
| **Subject** | Plan title |
| **Client** | Client name |
| **Time** | Plan period/date |
| **Plan Status** | Draft, In Progress, Review, Completed, Archived |
| **Objective** | Main objective |
| **Revenue Next Year** | Planned revenue |
| **Margin** | Margin percentage |
| **Wallet Share** | Wallet share percentage |
| **Client Status** | Red, Yellow, or Green |
| **BCG Model** | Question marks, Stars, Dogs, Cash cows |
| **Custom Fields** | Any custom fields you added |

---

## Filtering Plans

Use the filter bar to narrow down the list:

| Filter | What It Does |
|--------|--------------|
| **Search** | Search by text (subject, client, etc.) |
| **Client** | Show plans for one client only |
| **Plan Status** | Draft, In Progress, Review, Completed, Archived |
| **BCG Model** | Question marks, Stars, Dogs, Cash cows |
| **Person In Charge** | Filter by assigned staff |
| **Has relation** | Plans with invoice or proposal |
| **Overdue Tasks** | Plans with overdue tasks |
| **Date from / to** | Filter by plan date range |

**To apply filters:**

1. Set the filters you want.
2. Click the **Search** (magnifying glass) button.

**To clear filters:**

- Click the **Clear** (X) button.

---

## Saving Filters

If you use certain filters often, you can save them:

1. Apply your filters.
2. Click **Save Filter**.
3. Enter a name (e.g., "My In-Progress Plans").
4. Click **Submit**.

**To use a saved filter:**

1. Open the **Saved Filters** dropdown.
2. Select the filter you want.
3. The table will refresh with that filter applied.

## Opening a Plan

Click the **Subject** or **ID** of a plan to open it. You will see the plan view with all tabs (Due Diligence, Team Information, Planning, etc.).

---

## Quick Actions (Top Buttons)

| Button | Description |
|--------|-------------|
| **New Account** | Create a new account plan |
| **Templates** | Manage plan templates |
| **Kanban View** | Switch to Kanban board view |
| **Settings** | Open module settings (admin only) |

---

## Export to Excel

To export the current table (with filters applied) to Excel:

1. Apply any filters you want.
2. Use the **Export Excel** option (usually in the table menu or toolbar).
3. An Excel file (.xlsx) will be downloaded.
