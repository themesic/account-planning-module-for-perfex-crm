# Dashboard Widget

The Account Planning module adds a widget to the Perfex CRM dashboard. It gives you a quick overview of your account plans without opening the module.

---

## What the Widget Shows

| Metric | Description |
|--------|-------------|
| **Draft** | Number of plans in Draft status |
| **In Progress** | Number of plans in progress |
| **Completed** | Number of completed plans |
| **Overdue Tasks** | Total overdue tasks across all plans |
| **Client Status** | Count of Red, Yellow, Green clients |
| **Revenue vs Plan** | Planned revenue vs actual (invoice) revenue |
| **At-risk accounts** | Plans with Red status and overdue tasks |
| **View All** | Link to the full Account Planning list |

---

## When the Widget Appears

- The widget appears only if:
  - The module is **enabled**
  - You have **View** permission for Account Planning
- It is usually in the dashboard grid (e.g., one of the panels).

---

## Using the Widget

- **Quick check** – See how many plans need attention
- **At-risk link** – Click "At-risk accounts" to filter plans that are Red and have overdue tasks
- **Revenue comparison** – See if you're ahead or behind plan
