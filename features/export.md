The Account Planning module lets you export data in several formats for reporting, sharing, and archiving.

---

## Export PDF

Export a single account plan as a PDF document.

**How to:**

1. Open the account plan.
2. Click the **dropdown menu** (caret) next to the plan title.
3. Select **Export PDF**.
4. A new tab or window opens with the PDF.
5. Use your browser's print or save option to download.

**Use cases:**

- Share with clients in meetings
- Attach to emails
- Archive for compliance

---

## Export Excel

Export the account plans table to Excel (.xlsx).

**How to:**

1. Go to **Account Planning** (main list).
2. Apply any filters you want (only exported rows will be included).
3. Use the **Export Excel** option (in the table menu or toolbar).
4. An Excel file downloads.

**Use cases:**

- Analyze plans in a spreadsheet
- Create custom reports
- Import into other tools

---

## Export MindMap

The Planning tab includes a MindMap editor. You can export it as:

### PNG (Image)

1. Open the plan and go to the **Planning** tab.
2. Create or edit the MindMap (LAMP icon).
3. Use the **Export MindMap** or **Export as PNG** option.
4. An image file downloads.

### SVG (Vector)

1. Same as above.
2. Use **Export as SVG**.
3. An SVG file downloads (scalable, good for print).

**Use cases:**

- Include in presentations
- Print for workshops
- Share strategy visually
